O: Object-Oriented Programming

Explanation (English)

R: Recursion

Explanation (English)

This example defines a recursive function to compute the factorial of a number.


def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

print(factorial(5))

ব্যাখ্যা (বাংলা)

এই উদাহরণে একটি recursive ফাংশন ব্যবহার করে factorial হিসাব করা হয়েছে।

    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

Z: Zip Function

Explanation (English)

This example demonstrates the built-in zip() function to combine lists.

    print(num, letter)

ব্যাখ্যা (বাংলা)

এই উদাহরণে zip() ফাংশন ব্যবহার করে দুটি তালিকা থেকে যুগ্ম উপাদান তৈরি করা হয়েছে।

    print(num, letter)
