A: Animations

Explanation (English)

This code demonstrates a simple CSS animation that changes a div’s background color from red to yellow.


@keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}
div {
  width: 100px;
  height: 100px;
  animation: example 5s infinite;
}

ব্যাখ্যা (বাংলা)

এই কোডটি একটি সহজ CSS এনিমেশন দেখায় যা একটি div এর ব্যাকগ্রাউন্ড রঙকে লাল থেকে হলুদে পরিবর্তন করে।


@keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}
div {
  width: 100px;
  height: 100px;
  animation: example 5s infinite;
}

B: Box Model

Explanation (English)

This example illustrates the CSS box model, including padding, border, and margin.

  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

ব্যাখ্যা (বাংলা)

এই উদাহরণে CSS Box Model দেখানো হয়েছে, যেখানে প্যাডিং, বর্ডার এবং মার্জিন রয়েছে।


div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

  /* Blue color inherited */
}

M: Media Queries

Explanation (English)

This example shows how to use a media query to change styles based on screen width.

এই উদাহরণে মিডিয়া কুয়েরি ব্যবহার করে স্ক্রিনের চওড়ার উপর ভিত্তি করে স্টাইল পরিবর্তন দেখানো হয়েছে।


@media (max-width: 600px) {
  .responsive {
    font-size: 14px;
  }
}
