A: Arrays

Explanation (English)

Arrays are used to store multiple values in a single variable. 


// Creating an array of fruits
let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits);

ব্যাখ্যা (বাংলা)

অ্যারে একটি ভেরিয়েবলে একাধিক মান সংরক্ষণ করতে ব্যবহৃত হয়। 


// ফলের একটি অ্যারে তৈরি করা
let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits);

B: Booleans

Explanation (English)

Booleans represent two values: true or false. 


let isAvailable = true;
console.log(isAvailable);

ব্যাখ্যা (বাংলা)

বুলিয়ান শুধুমাত্র দুইটি মান ধারণ করে: true অথবা false. 


let isAvailable = true;
console.log(isAvailable);

C: Conditionals

Explanation (English)

Conditional statements execute different actions based on conditions. 


let age = 20;
if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

ব্যাখ্যা (বাংলা)

শর্তসাপেক্ষ বিবৃতি শর্ত অনুযায়ী বিভিন্ন কাজ করে। 


let age = 20;
if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

F: Functions

Explanation (English)

Functions are reusable blocks of code designed to perform a specific task. 


function greet(name) {
  return "Hello, " + name + "!";
}
console.log(greet("Alice"));

ব্যাখ্যা (বাংলা)

ফাংশনগুলি পুনঃব্যবহারযোগ্য কোড ব্লক যা নির্দিষ্ট কাজ করে। 


function greet(name) {
  return "Hello, " + name + "!";
}
console.log(greet("Alice"));

G: Generators

Explanation (English)

Generators are functions that can be paused and resumed, using the yield keyword. 


function* generatorExample() {
  yield 1;
  yield 2;
  yield 3;
}
const gen = generatorExample();
console.log(gen.next().value); // 1

ব্যাখ্যা (বাংলা)

জেনারেটর এমন ফাংশন যা থামানো এবং পুনরায় চালু করা যায়, yield কীওয়ার্ড ব্যবহার করে। 


function* generatorExample() {
  yield 1;
  yield 2;
  yield 3;
}
const gen = generatorExample();
console.log(gen.next().value); // 1

H: Hoisting

Explanation (English)

Hoisting allows variable and function declarations to be moved to the top of their scope. 


console.log(hoistedVar); // undefined
var hoistedVar = "I am hoisted";

ব্যাখ্যা (বাংলা)

হোইস্টিংয়ের ফলে ভেরিয়েবল ও ফাংশনের ডিক্লারেশন তাদের স্কোপের শীর্ষে সরিয়ে নেওয়া হয়। 


console.log(hoistedVar); // undefined
var hoistedVar = "I am hoisted";

I: IIFE

Explanation (English)

IIFE is a function that runs as soon as it is defined. 


(function() {
  console.log("IIFE executed!");
})();

ব্যাখ্যা (বাংলা)

IIFE এমন একটি ফাংশন যা সংজ্ঞায়িত হওয়ার সাথে সাথেই চালু হয়। 


(function() {
  console.log("IIFE executed!");
})();

J: JSON

Explanation (English)

JSON is a format for storing and transporting data. 


// Converting an object to JSON
const person = { name: "Alice", age: 25 };
const jsonStr = JSON.stringify(person);
console.log(jsonStr);

ব্যাখ্যা (বাংলা)

JSON হল ডেটা সংরক্ষণ ও পরিবহনের একটি ফরম্যাট। 


// একটি অবজেক্টকে JSON এ রূপান্তর করা
const person = { name: "Alice", age: 25 };
const jsonStr = JSON.stringify(person);
console.log(jsonStr);

K: Keywords

Explanation (English)

Keywords are reserved words that have special meaning in JavaScript (e.g., let, const, function). 


let a = 10;
const b = 20;
function add(x, y) {
  return x + y;
}

ব্যাখ্যা (বাংলা)

কীওয়ার্ড হল রিজার্ভড শব্দ যেগুলো জাভাস্ক্রিপ্টে বিশেষ অর্থ বহন করে (যেমন, let, const, function)। 


let a = 10;
const b = 20;
function add(x, y) {
  return x + y;
}

L: Loops

Explanation (English)

Loops are used to execute a block of code repeatedly. 


for (let i = 0; i < 5; i++) {
  console.log(i);
}

ব্যাখ্যা (বাংলা)

লুপের মাধ্যমে কোডের একটি ব্লক বারবার চালানো হয়। 


for (let i = 0; i < 5; i++) {
  console.log(i);
}

M: Modules

Explanation (English)

Modules help in organizing code by splitting it into separate files. (Requires a module-supporting environment.) 


// In math.js
export function add(a, b) {
  return a + b;
}

// In main.js
import { add } from './math.js';
console.log(add(2, 3));

ব্যাখ্যা (বাংলা)

মডিউল কোডকে আলাদা ফাইলে ভাগ করে কোড সংগঠিত করতে সহায়তা করে। (মডিউল সমর্থনকারী পরিবেশে ব্যবহার করুন) 


// math.js এ
export function add(a, b) {
  return a + b;
}

// main.js এ
import { add } from './math.js';
console.log(add(2, 3));

let num = 15.6789;

P: Promises

Explanation (English)

Promises handle asynchronous operations. They represent a value that may be available now, later, or never. 


let promise = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Done!"), 1000);
});
promise.then(result => console.log(result));

ব্যাখ্যা (বাংলা)

প্রমিস আসিঙ্ক্রোনাস অপারেশনগুলো হ্যান্ডেল করে। এটি এমন একটি মান যা এখন, পরে বা কখনোই পাওয়া নাও যেতে পারে। 


let promise = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Done!"), 1000);
});
promise.then(result => console.log(result));

R: Regular Expressions

Explanation (English)

Regular expressions are patterns used to match character combinations in strings. 


let regex = /hello/i;

ব্যাখ্যা (বাংলা)

রেগুলার এক্সপ্রেশন এমন প্যাটার্ন যা স্ট্রিংয়ের মধ্যে ক্যারেক্টার কম্বিনেশন ম্যাচ করতে ব্যবহৃত হয়। 


let regex = /hello/i;
console.log(regex.test("Hello World")); // true

Explanation (English)

Template literals allow embedded expressions and multi-line strings using backticks.

ব্যাখ্যা (বাংলা)

U: Undefined & Null

Explanation (English)

undefined means a variable has been declared but not assigned a value, while null represents no value. 


let a;
console.log(a); // undefined

let b = null;
console.log(b); // null

ব্যাখ্যা (বাংলা)

undefined নির্দেশ করে যে ভেরিয়েবল ঘোষণা করা হয়েছে কিন্তু কোন মান দেওয়া হয়নি, আর null কোনো মানের অভাব নির্দেশ করে। 


let a;
console.log(a); // undefined

let b = null;
console.log(b); // null

V: Variables

Explanation (English)

Variables are used to store data values. Use let or const in modern JavaScript. 


let name = "Alice";
const age = 25;
console.log(name, age);

ব্যাখ্যা (বাংলা)

ভেরিয়েবল ডেটা মান সংরক্ষণে ব্যবহৃত হয়। আধুনিক জাভাস্ক্রিপ্টে let বা const ব্যবহার করুন। 


let name = "Alice";
const age = 25;
console.log(name, age);

Web Storage provides mechanisms like localStorage and sessionStorage to store data in the browser.

ব্যাখ্যা (বাংলা)

// localStorage ব্যবহার করে ডেটা সেট এবং গেট করা
localStorage.setItem("username", "Alice");
console.log(localStorage.getItem("username"));

X: XMLHttpRequest

Explanation (English)

XMLHttpRequest (XHR) is used to interact with servers via HTTP. 


let xhr = new XMLHttpRequest();
xhr.open("GET", "https://api.example.com/data", true);
xhr.onload = function () {
  if (xhr.status === 200) {
    console.log(xhr.responseText);
  }
};
xhr.send();

ব্যাখ্যা (বাংলা)

XMLHttpRequest (XHR) HTTP এর মাধ্যমে সার্ভারের সাথে যোগাযোগ করতে ব্যবহৃত হয়। 


let xhr = new XMLHttpRequest();
xhr.open("GET", "https://api.example.com/data", true);
xhr.onload = function () {
  if (xhr.status === 200) {
    console.log(xhr.responseText);
  }
};
xhr.send();

Y: Yield

Explanation (English)

The yield keyword is used within generators to pause and resume execution. 


function* numberGenerator() {
  yield 10;
  yield 20;
}
const gen = numberGenerator();
console.log(gen.next().value); // 10

ব্যাখ্যা (বাংলা)

yield কীওয়ার্ড জেনারেটরের ভিতরে ব্যবহৃত হয় এক্সিকিউশন থামানো ও পুনরায় চালানোর জন্য। 


function* numberGenerator() {
  yield 10;
  yield 20;
}
const gen = numberGenerator();
console.log(gen.next().value); // 10

Z: Zipping Arrays

Explanation (English)

Zipping arrays combines two arrays into one array of paired elements. 


const array1 = [1, 2, 3];
const array2 = ['a', 'b', 'c'];
const zipped = array1.map((value, index) => [value, array2[index]]);
console.log(zipped); // [[1, 'a'], [2, 'b'], [3, 'c']]

ব্যাখ্যা (বাংলা)

জিপিং Arrays মানে দুইটি অ্যারে থেকে যুগ্ম উপাদানের একটি নতুন অ্যারে তৈরি করা। 


const array1 = [1, 2, 3];
const array2 = ['a', 'b', 'c'];
const zipped = array1.map((value, index) => [value, array2[index]]);
console.log(zipped); // [[1, 'a'], [2, 'b'], [3, 'c']]
